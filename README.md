# React Music Player

A fully responsive, interactive music player built with React. Features include a dynamic equalizer, volume control, seek bar, animations, and continuous play of multiple royalty-free tracks.

## Features

- **Responsive Design:** Mobile-first and works beautifully across all device sizes.
- **Play Controls:** Play, pause, stop, forward, previous, and shuffle.
- **Volume Control:** Adjust the volume dynamically.
- **Seek Bar:** Interactive progress bar that allows scrubbing to any part of the track.
- **Continuous Play:** Automatically moves to the next song unless stopped.
- **Animated Artwork & Titles:** Subtle animations on album art and song titles.
- **Equalizer:** Real-time animated bars reacting to the music.
- **Royalty-Free Music:** Tracks sourced from royalty-free YouTube creator music.
- **Mobile-Friendly:** Touch-friendly controls and adaptive layout.

## Tech Stack

- React
- Tailwind CSS
- Framer Motion (for optional animations)
- React Icons
